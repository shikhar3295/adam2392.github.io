VirtualEnv: 15.1.0 VirtualEnvWrapper

Basics (Create, Delete):

  1. Create For Virtualenvwrapper: Run the command 'mkvirtualenv '

For Virtualenv: Run command virtualenv './venv/' inside your project directory

  1. Delete a Virtualenv For Virtualenvwrappper: In order to delete a virtual environment, all you need to do is to remove it recursively from your ~/.virtualenvs directory with 'sudo rm -rf '.

For Virtualenv: Go to the directory of your project and delete the /venv/ directory.
